**`compute_percentile(sorted_values: list[int], percentile: float) -> int`** — Pure function.
- Takes a **pre-sorted** list of integers and a percentile (0–100).
- Returns the percentile value using **linear interpolation** per D-01.
- Algorithm (standard linear interpolation, matches numpy default `method='linear'`):
  ```python
  def compute_percentile(sorted_values: list[int], percentile: float) -> int:
      """Compute a percentile using linear interpolation.

      Args:
          sorted_values: Pre-sorted list of integer values (ascending).
          percentile: Percentile to compute (0-100).

      Returns:
          Interpolated percentile value as int.
      """
      if not sorted_values:
          return 0
      n = len(sorted_values)
      if n == 1:
          return sorted_values[0]
      # Linear interpolation (matches numpy default)
      rank = (percentile / 100.0) * (n - 1)
      low = int(rank)
      high = min(low + 1, n - 1)
      frac = rank - low
      return int(sorted_values[low] + frac * (sorted_values[high] - sorted_values[low]))
  ```

**`aggregate_span_statistics(traces: list[dict[str, Any]]) -> list[OperationStats]`** — Pure function.
- Takes a list of raw Jaeger trace dicts (each is `data[i]` from `GET /api/traces`).
- Iterates all spans across all traces. For each span, extracts `operationName` and `duration` (microseconds) and checks `is_error` using `span_is_error()`.
- Groups by `operationName`. For each operation group:
  - `count` = number of spans
  - Sort durations, compute `p50_duration_us`, `p95_duration_us`, `p99_duration_us` using `compute_percentile()`
  - `error_count` = number of spans where `span_is_error()` is True
  - `error_rate` = `error_count / count` (float; 0.0 if count is 0 — but count can't be 0 in a group)
- Returns a `list[OperationStats]` sorted alphabetically by `operation` name per OpenCode's discretion.
- If `traces` is empty or all traces have no spans, return `[]` per D-03.

Implementation:
```python
def aggregate_span_statistics(traces: list[dict[str, Any]]) -> list[OperationStats]:
    """Aggregate per-operation latency and error stats across traces.

    Args:
        traces: List of raw Jaeger trace dicts (each containing 'spans').

    Returns:
        List of :class:`OperationStats`, one per distinct operation,
        sorted alphabetically by operation name.
    """
    # Collect durations and error flags per operation
    ops: dict[str, dict[str, Any]] = {}
    for trace in traces:
        for span in trace.get("spans") or []:
            op = span.get("operationName", "")
            if op not in ops:
                ops[op] = {"durations": [], "error_count": 0}
            ops[op]["durations"].append(span.get("duration", 0))
            if span_is_error(span):
                ops[op]["error_count"] += 1

    stats: list[OperationStats] = []
    for op in sorted(ops):
        data = ops[op]
        durations = sorted(data["durations"])
        count = len(durations)
        error_count = data["error_count"]
        stats.append({
            "operation": op,
            "count": count,
            "p50_duration_us": compute_percentile(durations, 50),
            "p95_duration_us": compute_percentile(durations, 95),
            "p99_duration_us": compute_percentile(durations, 99),
            "error_count": error_count,
            "error_rate": error_count / count if count else 0.0,
        })
    return stats
```
