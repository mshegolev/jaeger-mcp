  <action>
Add two new functions to shaping.py:

**`span_match_key(span, processes, span_map)`** — Pure function. Extracts the (operationName, serviceName, parentOperation) tuple per D-01.
- `span`: raw Jaeger span dict
- `processes`: Jaeger processes map
- `span_map`: dict mapping spanID → span dict (for resolving parent operation name)
- Resolve serviceName from processID → processes[pid].serviceName
- Resolve parentOperation: find CHILD_OF ref → look up parent span in span_map → get parent's operationName. None if root.
- Return: `tuple[str, str, str | None]`

**`compare_traces_diff(trace_a, trace_b)`** — Pure function. Takes two raw Jaeger trace dicts.
- Extract spans and processes from each trace
- Build span_map for each trace (spanID → span)
- Compute match key for every span in each trace using span_match_key()
- Group spans by match key. For duplicate keys (multiple spans with same key), collect as lists preserving order.
- Pair duplicates by position order (per D-02): first occurrence in A pairs with first in B, etc.
- For each key:
  - Only in A → removed_spans (as MatchedSpanSummary)
  - Only in B → added_spans (as MatchedSpanSummary)
  - In both → compare duration and tags:
    - If duration differs OR tags differ → changed_spans (as ChangedSpan with deltas)
    - If identical → increment unchanged_count
- Tag diff: use span_tags_flat() on both spans. tags_added = keys in B not in A. tags_removed = keys in A not in B. tags_changed = keys in both where value differs (value is the B value).
- Return CompareTracesOutput with trace_id_a, trace_id_b from trace dicts' traceID field.
