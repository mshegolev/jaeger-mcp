**client.py — full rewrite of HTTP internals:**

1. Remove imports: `import requests`, `import urllib3`, `from requests.adapters import HTTPAdapter`, `from urllib3.util.retry import Retry`. Replace with `import httpx` and `import asyncio`.

2. Replace `threading.Lock` with `asyncio.Lock` for `_cache_lock`. Change `import threading` to `import asyncio`.

3. In `__init__`:
   - Remove `self.session = requests.Session()` and all session configuration.
   - Remove the `Retry` + `HTTPAdapter` mount logic entirely.
   - Store retry config as instance attrs: `self._retry_total = retries`, `self._retry_backoff = 1`, `self._retry_status_codes = {429, 500, 502, 503, 504}`.
   - Store `self._cache_lock = asyncio.Lock()`.
   - Do NOT create httpx.AsyncClient in __init__ — store config only. The client will be lazily created via `_ensure_client()` since AsyncClient needs an async context.
   - Build headers dict: `self._headers = {"Accept": "application/json", "User-Agent": "jaeger-mcp"}`.
   - If token: add `"Authorization": f"Bearer {self.token}"` to headers.
   - Store `self._auth = (self.username, self.password) if self.username and self.password else None` for Basic auth (httpx uses `auth=` param).
   - Store `self._client: httpx.AsyncClient | None = None`.
   - Keep the `urllib3.disable_warnings` call replaced by nothing — httpx handles this via `verify=False` without warnings. Remove the `urllib3` import entirely.

4. Add `async def _ensure_client(self) -> httpx.AsyncClient`:
   ```python
   if self._client is None:
       self._client = httpx.AsyncClient(
           headers=self._headers,
           auth=self._auth,
           verify=self.ssl_verify,
           timeout=httpx.Timeout(self.timeout),
           follow_redirects=True,
       )
   return self._client
   ```

5. Replace `_request` with `async def _request(self, method, endpoint, *, params=None) -> httpx.Response`:
   - Call `client = await self._ensure_client()`
   - Use `response = await client.request(method=method, url=url, params=params)`
   - Keep the timing log (method, url, status, ms).
   - Call `response.raise_for_status()`.
   - Return response.

6. Add retry logic as a wrapper method `async def _request_with_retry(self, method, endpoint, *, params=None) -> httpx.Response`:
   ```python
   last_exc = None
   for attempt in range(self._retry_total + 1):
       try:
           return await self._request(method, endpoint, params=params)
       except httpx.HTTPStatusError as exc:
           if exc.response.status_code not in self._retry_status_codes:
               raise
           last_exc = exc
           if attempt < self._retry_total:
               delay = self._retry_backoff * (2 ** attempt)
               await asyncio.sleep(delay)
       except (httpx.ConnectError, httpx.ConnectTimeout) as exc:
           last_exc = exc
           if attempt < self._retry_total:
               delay = self._retry_backoff * (2 ** attempt)
               await asyncio.sleep(delay)
   raise last_exc
   ```

7. Convert cache methods to async:
   - `async def _cache_get(self, key)`: use `async with self._cache_lock:`
   - `async def _cache_set(self, key, value)`: use `async with self._cache_lock:`

8. Convert `get` to `async def get(self, endpoint, params=None) -> Any`:
   - Same cache logic but with `await self._cache_get()` and `await self._cache_set()`.
   - Call `response = await self._request_with_retry("GET", endpoint, params=params)`.
   - Use `response.json()` (httpx json() is sync, not a coroutine).
   - Check `if not response.content:` for empty body.

9. Convert `close` to `async def close(self) -> None`:
   - `if self._client is not None: await self._client.aclose()`.
   - Keep credential zeroing: `self.token = ""`, `self.username = ""`, `self.password = ""`.
